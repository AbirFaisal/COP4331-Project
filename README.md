### Building, Testing, and Running:

System Requirements:
- Java Runtime 20

Build Requirements:
- OpenJDK 20
- Maven

Build System: Maven

Build Instructions

- Install Maven if not installed already

Compile and Package:
- Switch to directory `TTT3D/`
- Run: `mvn package -f pom.xml`

Running:
- Switch to `target/` directory
- Run `java -jar TTT3D.jar`
